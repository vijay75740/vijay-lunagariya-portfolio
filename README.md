## üöÄ Quick Start

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/vijay-lunagariya-portfolio.git
```

2. Navigate to the project directory:
```bash
cd vijay-portfolio
```

3. Open `index.html` in your browser:
   - Double-click the file, or
   - Use a local server (recommended):
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx http-server
   ```

4. Visit `http://localhost:8000` in your browser

## üìù Customization

### Update Personal Information

Edit the following sections in `index.html`:

1. **Contact Information** (lines ~450-500):
   - Email address
   - Phone number
   - LinkedIn URL
   - Location

2. **Projects** (lines ~350-450):
   - Add or modify project cards
   - Update descriptions
   - Add project links

3. **Skills** (lines ~300-350):
   - Add or remove skill tags
   - Update skill categories

4. **About Section** (lines ~250-300):
   - Modify bio text
   - Update statistics

### Styling

All styles are in the `<style>` section of `index.html`. Key CSS variables are defined at the top:

```css
:root {
    --bg-primary: #0a0e27;
    --accent-primary: #00d9ff;
    --accent-secondary: #7c3aed;
    /* Modify these to change the color scheme */
}
```
